# 🛍️ E-Commerce Flutter App

A modern, responsive E-Commerce application built using Flutter. This app features clean UI, network-based product and category fetching, and seamless user interaction across all major screens.

## ✨ Features
- Splash screen with app branding
- User Authentication (Login/Register) with form validation
- API integration to fetch categories and products
- Home screen with hero banner, category grid, and search functionality
- Custom reusable widgets for scalable UI
- Modern UI following clean white and blue color scheme
- Responsive design supporting both light and dark themes

## 📸 Screens
- Splash Screen
- Login/Register
- Home (with Categories & Promotions)
- Product List (based on category)
- Bottom Navigation Bar (Home, Search, Favorites, Profile)

## 🛠️ Technologies Used
- Flutter & Dart
- REST API Integration
- `dot_curved_bottom_nav` package for bottom nav
- Google Fonts (Inter)
- Custom Themes and Reusable Widgets
